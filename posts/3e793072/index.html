基本语法

标题

以下分别代表一至五级标题：

#h1
##h2
###h3
####h4
#####h5

引用

在段落等内容前使用 > 符号，可以将这段内容标记为引用的内容：

>引用内容
>>二级引用

引用内容

二级引用

列表

无序列表

* 可以作为标记
+ 也可以
- 同样可以
  • 可以作为标记
  • 也可以
  • 同样可以

有序列表

1. 以数字和`. `开始；
3. 数字的顺序并不会影响生成的列表序列；
  1. 以数字和 . 开始；
  2. 数字的顺序并不会影响生成的列表序列；

嵌套的列表

1. 第一层
    + 1-1
    + 1-2
2. 无序列表和有序列表可以随意嵌套
    1. 2-1
    2. 2-2
  1. 第一层
    • 1-1
    • 1-2
  2. 无序列表和有序列表可以随意嵌套
    1. 2-1
    2. 2-2

代码

代码块

在代码的头部和尾部用 ``` 包围起来，需要高亮的话，在 ``` 包围的代码块头部标注语法的名字：

s = 'Python syntax highlighting'
for chr in s:
    print (s)

行内代码

通过 插入行内代码：

print(Hello world)

分隔线

一行中使用三个即以上的 *、-、_来添加分隔线：

****

超链接

需要显示的文本为 [] 的内容，小括号中的链接可以是本地也可以是网络

[Google](https://google.com)

Google

图片

语法与超链接基本一致，只是需要在前面加一个 !：

![title](https://cdn.pixabay.com/photo/2012/10/26/00/45/senegal-pechlibelle-62964_960_720.jpg)

强调

粗体

使用 ** ** 或 __ __ 包括的文本会转化成粗体

斜体

使用 * * 或 _ _ 包括的文本会转化为斜体

字符的转义

当需要显示一些特殊的字符的时候，需要使用反斜线 \ 来进行转义：

这是一个*转义*的_例子_
这是一个\*转义\*的\_例子\_

这是一个转义的例子

这是一个*转义*的_例子_

进阶语法

删除线

这是~~删除线~~

这是删除线

下划线

Markdown 原生并无下划线语法，可以通过 HTML 来实现：

<u>这是下划线</u>

这是下划线

表格

单元格和表头

使用 | 来分隔不同单元格，使用 - 来分隔表头和其他行：

| name  | age  |
| ----- | ---- |
| Leo   | 16   |
| Virgo | 18   |
nameage
Leo16
Virgo18

对齐

在表头下方的分隔线标记加入 :，可更换下方单元格的对齐方式：

| left | center | right |
| ---- | :----: | ----: |
| left | center | right |
leftcenterright
leftcenterright

- [x] 唱歌

st=>start: Start
e=>end: End
op=>operation: My Operation

st->op->cond
cond(yes)->e
cond(no)->op

时序图

Note right of Bob: Bob thinks
